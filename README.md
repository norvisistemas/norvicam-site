# NorviCam Landing Page

Landing page estatica para divulgar o NorviCam no Instagram e entregar o APK de teste.

## Como testar localmente

Abra `index.html` no navegador.

## Como hospedar no Firebase Hosting

1. Instale o Firebase CLI.
2. Rode `firebase login`.
3. Rode `firebase init hosting`.
4. Escolha esta pasta como pasta publica.
5. Rode `firebase deploy`.
